This project aims to create a short and efficient video creator with ffmpeg and Typecript.
Agents icons are in icons/
Agents PiperTTS voices are in voices/

Before committing, please `bun lint` to check errors.
You can fix them with `bun lint --fix --unsafe`.
Please format your code with `bun format`.
Finally check your code with `bun check` to ensure everything is working correctly.
Build the project with `bun build` and run it with `bun run start`.
